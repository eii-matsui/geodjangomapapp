機能概要

現在地自動取得

Geolocation API で現在の位置情報を取得してGoogleMapにマーカーを表示。

避難所をマッピング

G空間情報センターで公開されている指定緊急避難場所をGoogleMapにマーカー表示。

最短経路の表示

避難所のマーカーをクリックすると現在地からの最短経路を表示。

移動距離の表示

最短経路と合わせて移動距離(メートル)を表示。
